# Article images live here

Drop your PNG, JPG, or SVG files into this folder.

To reference them in a markdown article, use:

    ![Caption text](/articles/your-image-name.png)

Tips:
- Use lowercase names with hyphens (e.g. `tesla-iv-chart.png`)
- Optimize images before adding (TinyPNG.com is free and good)
- Save Claude-generated charts as PNG for best clarity
